React-TicTacToe & AWS amplify

15/05/20

重點:

1. 使用virtual DOM，增加效能。

2. 使用 JSX語法，方便用javascript建立UI。

3. 組件化，方便重用和維護。

4. Immutability（不可變性），可簡化複雜功能，容易偵測改變。

5. AWS Amplify 是一個開發平台,可用於建立安全、可擴展的行動和 Web 應用程式。

基本syntax

ReactDOM.render(react-element,html-element);

ReactDOM.render( < h1>Hello, world!< /h1>, document.getElementById('root'));

Virtual DOM

使用javascript 模擬html dom tree，將改變暫存在javascript dom tree上，使用Diffing 演算法比較2個tree，將更新套用到html dom tree上。

JSX語法

JSX是 JavaScript 的語法擴充，可以更直觀描寫UI外觀。

const element = < h1>Hello World!!< /h1>;

Components

將UI拆細成多個獨立可重用的Components，Components會return react element，分作function components 和 class components，class components有自己的State(local variable)。

Immutability（不可變性）

類似function program概念，唯讀input，相同input相同output，function內不改變variable。

AWS Amplify

可用於部署和託管 ReactJS 應用程式，另外有執行後端程式等功能。

建立react app 並架設到AWS Amplify

1. 建立一個GitHub Repository, 參考: https://guides.github.com/activities/hello-world/

2. 建立react app, 參考: https://zh-hant.reactjs.org/docs/getting-started.html

3. 由於react app 無止架設到GitHub Page, 建立AWS Amplify並架設, 參考: https://zh-hant.reactjs.org/tutorial/tutorial.html
